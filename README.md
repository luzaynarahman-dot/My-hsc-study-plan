# 🚀 Study Vault | HSC 2026 Revision Tracker

A minimalist, high-performance web dashboard designed to manage the massive HSC syllabus (88+ chapters) within a 45-day revision window. This project combines my passion for **front-end development** with the discipline needed for academic excellence.

## 🌟 The "Why" (Motivation)
About 2-3 months before the HSC exams, the vastness of the syllabus (Physics, Math, Biology, Chemistry, ICT) started feeling overwhelming. As a coding enthusiast, I wanted to build a tool that wouldn't just be a static list, but a **visual motivation engine**. 

I designed this to:
* Maintain a **15-day Math Sprint** (10 chapters of 1st Paper).
* Follow a strict **Monthly Biology Focus** (Botany in April, Zoology in May).
* Ensure **Daily Consistency** with a structured 5:00 AM to Night schedule.

## 🛠️ Features
- **Interactive Checklist:** Track progress across all 88 chapters.
- **Subject-Wise Segmentation:** Separate views for 1st and 2nd papers.
- **Dynamic Progress Bars:** Real-time visual feedback on how much of the syllabus is "slayed."
- **Premium Glassmorphism UI:** A clean, distraction-free dark interface optimized for mobile (Acode/TrebEdit).
- **User Impact:** Shared with friends to help them stay consistent; seeing the "checkmarks" fill up has turned stressful revision into a gamified challenge for all of us!

## 🚀 Tech Stack & Collaboration
* **Concept & UI Logic:** Designed by me (**Luzayna**) based on my personal study requirements and aesthetic sense.
* **Languages:** HTML5, CSS3 (Custom Properties & Flexbox), and Vanilla JavaScript.
* **Environment:** Developed entirely on **Android** using mobile code editors.
* **AI-Assisted Development:** I used AI (DeepSeek/Gemini) as a **Creative Assistant** and "Code Partner" to refine the technical implementation and optimize the UI logic. 

## 💡 Philosophy
> "AI is the engine, but the student is the navigator." This project proves that with the right tools and a clear plan, even the most daunting syllabus can be conquered.
